# 🏥 Sistema de Gestão de Clínica (Full Stack)

## 📌 Descrição
Sistema web full stack desenvolvido para gerenciamento de clínicas, permitindo o controle de pacientes, médicos e consultas.

O sistema foi desenvolvido com Angular no front-end e ASP.NET Core Web API no back-end, com banco de dados PostgreSQL.

Este foi o meu primeiro projeto utilizando Angular, integrando front-end moderno com API REST.

## 🚀 Tecnologias
Front-end
- Angular
- Angular Material
- TypeScript
- HTML
- CSS
- Flex Layout

Back-end
- ASP.NET Core Web API
- C#
- Entity Framework Core
- SQL Server

## 🎯 Funcionalidades
- Cadastro de clínicas
- Cadastro de pacientes
- Cadastro de médicos
- Agendamento de consultas
- Edição e exclusão de registros
- Integração entre Angular e API
- Consumo de endpoints REST
- Validações de formulários
- Máscaras de telefone e CPF
- Busca de estados e cidades

## 🔗 Arquitetura do projeto
O sistema é dividido em duas partes:

- Front-end (Angular): responsável pela interface do usuário
- Back-end (Web API): responsável pelas regras de negócio e banco de dados

A comunicação entre eles é feita via HTTP (REST API).

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Desenvolvimento full stack
- Integração entre Angular e ASP.NET Core Web API
- Consumo de APIs REST
- Organização de front-end moderno com componentes
- Boas práticas de separação de camadas

Além disso, foi meu primeiro contato prático com Angular.
